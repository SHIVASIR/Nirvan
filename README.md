🌿 Nirmaan – Smart Internship & Opportunity Platform

Nirwaan is an intelligent platform designed to help students discover verified internships, jobs, and opportunities while protecting them from scams. It focuses on building a safe, AI-powered ecosystem for students in India.

📌 Overview

In today’s digital world, students often struggle to find genuine internships and fall victim to fake listings.
Nirwaan aims to solve this problem by combining:

🔍 Opportunity discovery
🛡️ Scam detection
🤖 AI-based recommendations

into a single, easy-to-use platform.

🚀 Key Features
🔎 Internship & Job Listings
Explore opportunities from multiple domains in one place
🛡️ Scam Detection System
Identify and flag suspicious or fake listings
🤖 AI-Powered Recommendations
Personalized suggestions based on user profile
📊 User-Friendly Dashboard
Clean interface for easy navigation
🔐 Secure Access System (Future Scope)
Authentication and profile management
